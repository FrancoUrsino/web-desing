¿Qué contenido se va a dar a lo largo de este curso?

Todo lo necesario para maquetar un sitio web tanto de HTML como de CSS desde lo más basico: que son las etiquetas, clases, id, la importancia de cada una, flexbox, grid, custom properties, animaciones, box model y mucho más!!. Ademas, todos los cursos tienen un proyecto para entregar al final y ver si lo aprendido está aplicado. Una vez aprobado se les dará un título de finalizacion y aprobación del curso;luego de finalizar el mismo, vas a poner crear tus propios sitios web y darles el diseño que más te guste
